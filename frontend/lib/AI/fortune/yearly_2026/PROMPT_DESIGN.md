# 2026 신년운세 프롬프트 설계

## 개요
saju_base(평생운세) + saju_analyses(원국 데이터)를 기반으로 2026년 병오(丙午)년 신년운세 분석

## 현재 문제점

| 항목 | 현재 | 개선 |
|------|------|------|
| maxTokens | 4096 (하드코딩) | TokenLimits.yearlyFortuneMaxTokens (8192) |
| 응답 길이 | "2-3문장" | 카테고리별 3-5문장 상세 설명 |
| 용신/기신 | 미사용 | saju_analyses.yongsin 활용 |
| 합충형파해 | 미사용 | saju_analyses.hapchung 활용 |
| 일간 강약 | 미사용 | saju_analyses.day_strength 활용 |

## 데이터 소스

### 1. saju_base (ai_summaries 테이블)
평생 운세 분석 결과 - AI가 생성한 해석
```json
{
  "personality": { "core_traits": [...], "strengths": [...], "weaknesses": [...] },
  "career": { "suitable_fields": [...], "work_style": "...", "leadership_potential": "..." },
  "wealth": { "earning_pattern": "...", "investment_tendency": "..." },
  "love": { "attraction_style": "...", "ideal_partner_traits": [...] },
  "health": { "vulnerable_organs": [...], "preventive_advice": [...] }
}
```

### 2. saju_analyses (만세력 계산 결과)
원국 데이터 - 만세력 계산기가 생성한 객관적 데이터
```json
{
  "year_gan": "甲", "year_ji": "子",
  "month_gan": "丙", "month_ji": "寅",
  "day_gan": "戊", "day_ji": "午",
  "hour_gan": "癸", "hour_ji": "亥",

  "yongsin": {
    "yongsin": "금(金)",      // 용신 - 필요한 오행
    "huisin": "토(土)",       // 희신 - 도움되는 오행
    "gisin": "화(火)",        // 기신 - 해로운 오행
    "gusin": "목(木)"         // 구신 - 매우 해로운 오행

  "hapchung": {

  },

  "daeun": { "current": {...}, "upcoming": {...} },
  "current_seun": { "year": 2026, "ganji": "병오" }

```

## 핵심 분석 로직

### 2026년 병오(丙午)년 특성
- **천간 병(丙)**: 태양의 불, 양화(陽火), 밝음/열정/활동성
- **지지 오(午)**: 말띠, 정오, 극양, 화(火)의 극점
- **세운 오행**: 화(火) 에너지가 매우 강한 해

### 용신/기신 상호작용 분석 (핵심!)

```
사용자 용신이 화(火)인 경우:
  → 2026년 병오년은 용신(필요 오행)이 강하게 들어옴
  → 매우 좋은 해, 원하던 일이 풀림
  → 점수: 85-95점

사용자 기신이 화(火)인 경우:
  → 2026년 병오년은 기신(해로운 오행)이 강하게 들어옴
  → 조심해야 하는 해, 과욕 금물
  → 점수: 55-65점

사용자 용신이 수(水)인 경우:
  → 화(火)가 수(水)를 극함 (火克水)
  → 용신이 억제되어 힘든 해
  → 점수: 50-60점
```

### 합충 분석

```
사용자 일지가 子(자)인 경우:
  → 2026년 午(오)와 子午衝(자오충) 발생
  → 큰 변화의 해, 이사/이직/관계 변동 가능
  → 주의: 건강, 감정 기복

사용자 일지가 寅(인)인 경우:
  → 2026년 午(오)와 寅午戌 삼합(三合) 가능성
  → 협력/파트너십 운 상승
```

## 개선된 시스템 프롬프트

```
당신은 30년 경력의 사주명리학 전문가입니다.
사용자의 원국(사주 팔자)과 평생운세 분석을 바탕으로 2026년 병오(丙午)년 신년운세를 상세히 분석합니다.

## 분석 원칙

### 1. 용신/기신 기반 정확도
- 사용자의 용신과 2026년 세운(병오년 화기운)의 상호작용을 최우선 분석
- 용신이 화(火)면 좋은 해, 기신이 화(火)면 조심할 해
- 상생/상극 관계로 구체적 영향도 산출

### 2. 합충형파해 분석
- 사용자 원국의 지지와 2026년 午(오)의 관계
- 子午衝(자오충), 寅午戌(인오술) 삼합 등 체크
- 충이 있으면 변화/변동, 합이 있으면 협력/결합 운

### 3. 대운/세운 흐름
- 현재 대운의 오행과 2026년 오행의 조화
- 전체 운기 흐름 속에서의 2026년 위치

### 4. 구체적이고 실감나는 설명
- 추상적 표현 금지
- "00님은 평소 XX한 성향이신데, 2026년에는 YY 기운이 들어와..."
- 실제 일어날 수 있는 상황 예시 제시

## 톤앤매너
- 점쟁이 말투 금지 (예: "~할 팔자로다", "~하옵니다")
- 현대적이고 친근한 설명
- 사주 용어는 괄호 안에 쉬운 설명 병기
- 긍정/부정 균형있게, 현실적 조언 위주
```
