# 2026 신년운세 프롬프트 설계 (v5.1)

## 개요
saju_base(평생운세) + saju_analyses(원국 데이터)를 기반으로 2026년 병오(丙午)년 신년운세 분석

## v5.1 변경사항

| 항목 | v5.0 | v5.1 |
|------|------|------|
| yearInfo 섹션 | 있음 | **제거** (중복/불필요) |
| overview 구조 | 단일 reading | **6개 서브필드** (30문장+) |
| maxTokens | 8192 | **15000** (상세 응답용) |
| 카테고리 reading | 6-8문장 | **12-15문장** 상세 |

## 핵심 원칙: overview 30문장 이상!

```
overview.opening (3-4문장)
+ ilganAnalysis (5-6문장)
+ yongshinAnalysis (5-6문장)
+ hapchungAnalysis (5-6문장)
+ sinsalAnalysis (4-5문장)
+ yearEnergyConclusion (5-6문장)
= 최소 30문장
```

## 2026년 병오(丙午)년 특성

### 천간 병(丙)
- **속성**: 양화(陽火), 태양의 불, 밝고 뜨거움
- **성격**: 열정, 활동성, 리더십, 화려함
- **작용**: 강렬하고 직접적인 에너지

### 지지 오(午)
- **속성**: 화(火), 말띠, 정오의 극양
- **성격**: 활발함, 열정, 진취성
- **작용**: 에너지 최고조, 도화(桃花) 기운

### 병오년 종합
- 천간과 지지 모두 화(火)로 에너지가 매우 강한 해
- 납음: 천하수(天河水) - 극양이 극음으로 전환
- 십이운성: 제왕(帝王) - 에너지 최고조 상태
- 신살: 도화(桃花) - 매력과 인연의 기운

## 일간별 화(火) 영향 (십성)

| 일간 | 화(火)가 되는 십성 | 2026년 영향 |
|------|------------------|------------|
| 갑(甲), 을(乙) | **식상** | 표현/창작 활발, 에너지 소진 주의 |
| 병(丙), 정(丁) | **비겁** | 에너지 넘침, 경쟁/충돌 주의 |
| 무(戊), 기(己) | **인성** | 배움/귀인 좋음, 의존 주의 |
| 경(庚), 신(辛) | **관살** | 성장/압박, 스트레스 관리 필수 |
| 임(壬), 계(癸) | **재성** | 재물 기회 많음, 과욕 주의 |
