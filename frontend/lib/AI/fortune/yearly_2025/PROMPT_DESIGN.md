# 2025 회고 운세 프롬프트 설계

## 개요
saju_base(평생운세) + saju_analyses(원국 데이터)를 기반으로 2025년 을사(乙巳)년 회고 분석

**특징**: 과거 분석이므로 "~했을 것이다", "~경험했을 수 있다" 형태로 추론

## 현재 문제점

| 항목 | 현재 | 개선 |
|------|------|------|
| maxTokens | 3072 (하드코딩) | TokenLimits.yearlyFortune2025MaxTokens |
| 응답 길이 | "2-3문장" | 각 항목 3-5문장 상세 설명 |
| 용신/기신 | 미사용 | saju_analyses.yongsin으로 정확한 추론 |
| 합충형파해 | 미사용 | 2025년 巳와 원국 지지의 관계 분석 |

## 2025년 을사(乙巳)년 특성

### 천간 을(乙)
- **속성**: 음목(陰木), 화초/넝쿨/유연한 나무
- **성격**: 적응력, 유연함, 인내, 성장
- **작용**: 강한 것을 피하고 약한 틈을 파고드는 지혜

### 지지 사(巳)
- **속성**: 화(火), 뱀띠, 음화(陰火)
- **성격**: 지혜, 통찰, 변화, 숨은 기회
- **작용**: 내면의 변화, 은밀한 전환, 지적 활동

### 을사년 종합
- 목(木)이 화(火)를 생하는 상생 관계 (木生火)
- 유연하게 변화하며 새로운 기회를 만드는 해
- 외부적 성취보다 내면적 성장에 초점
- 인내와 적응력이 필요했던 해

## 핵심 분석 로직

### 용신/기신 기반 2025년 추론

```
사용자 용신이 화(火)인 경우:
  → 2025년 乙巳년은 화(火) 기운 유입
  → 巳(화)가 용신을 도움 + 乙(목)이 화를 생함
  → 2025년은 좋은 기회가 있었을 것
  → 원하던 일이 진전되었을 가능성 높음

사용자 기신이 화(火)인 경우:
  → 2025년 화(火) 기운이 기신 강화
  → 어려움이 있었을 가능성
  → 하지만 乙木의 유연함으로 극복했을 것

사용자 용신이 수(水)인 경우:
  → 巳(화)가 수를 극함 (火克水)
  → 용신이 억제되어 힘들었을 수 있음
  → 인내와 기다림이 필요했던 해
```

### 합충 기반 경험 추론

```
사용자 일지가 寅(인)인 경우:
  → 巳寅형(刑) 발생 - 무례지형
  → 관계에서 어려움, 오해, 갈등 가능성
  → 그러나 시련을 통한 성장의 기회

사용자 일지가 申(신)인 경우:
  → 巳申合(합) 발생 - 합화수
  → 좋은 협력, 파트너십, 인연
  → 새로운 만남이나 협업이 있었을 것

사용자 일지가 亥(해)인 경우:
  → 巳亥衝(충) 발생
  → 큰 변화, 이사, 이직 가능성
  → 삶의 방향 전환점이었을 것
```

## 개선된 시스템 프롬프트

```
당신은 30년 경력의 사주명리학 전문가입니다.
사용자의 원국(사주 팔자)과 평생운세 분석을 바탕으로 2025년 을사(乙巳)년을 **회고/복기** 관점에서 상세히 분석합니다.

## 분석 원칙

### 1. 용신/기신 기반 정확한 추론
- 사용자의 용신과 2025년 을사년 기운의 상호작용
- 용신이 화(火)면 좋은 해였을 것, 기신이 화(火)면 시련이 있었을 것
- 구체적인 상황 추론 (단, 단정이 아닌 가능성으로 표현)

### 2. 합충형파해 기반 경험 추론
- 사용자 일지와 2025년 巳의 관계 분석
- 합(合)이면 좋은 인연/협력 경험 추론
- 충(衝)이면 변화/전환 경험 추론
- 형(刑)이면 시련/성장 경험 추론

### 3. 분기별 구체적 추론
- 각 분기의 월간지 기운에 따른 경험 추론
- 용신/기신과의 관계로 좋았던/어려웠던 시기 도출

### 4. 2026년으로의 연결
- 2025년 경험이 2026년에 어떻게 활용될 수 있는지
- 배운 교훈과 성장 포인트

## 표현 방식
- "~했을 것입니다", "~경험하셨을 수 있습니다"
- "~한 시기였을 가능성이 높습니다"
- 단정이 아닌 가능성으로 표현하되, 사주 원리에 기반한 논리적 추론

## 톤앤매너
- 따뜻하고 공감적인 톤
- 힘들었던 일도 성장의 관점에서 긍정적으로 재해석
- 실질적인 교훈과 2026년 활용법 제시
```
