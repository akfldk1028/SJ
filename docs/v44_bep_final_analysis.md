## 8. 리스크와 대응

### 8.1 가장 큰 리스크: 클릭이 안 나온다면?

```
현재 CTR 실측: 4.89% (38클릭 / 777노출)
= 노출 100번 중 5번 클릭
= 하루 인라인 2-3개 노출 → 0.1~0.15회 클릭

문제: 인라인 노출만으로는 클릭 1/일 미달 가능
해결: 토큰 소진 배너에서 클릭 유도 (필수 광고)
  → 무료 20K 소진 → 배너 → 클릭 → 이건 CTR 거의 100%
  → 대화 이어가려면 반드시 클릭

결론: 소진 배너 클릭은 "사실상 강제" → 최소 1회/일 보장
```

### 8.2 CPC가 $0.05보다 낮다면?

```
CPC $0.03 시:
  소진 click 수익: $0.030
  AI 비용: $0.008
  순이익: +$0.022/cycle → 여전히 흑자

CPC $0.01 시 (최악):
  소진 click 수익: $0.010
  AI 비용: $0.008
  순이익: +$0.002/cycle → 겨우 흑자

BEP: $0.251 / ($0.002 + $0.015) = 14.8일 → 여전히 2주 내 BEP

→ CPC $0.01 (거의 불가능한 최악) 에서도 흑자
→ CPC가 얼마든 AI 비용($0.008)보다 높으면 이득
```

### 8.3 AI 가격 인상 리스크

```
Gemini Flash가 2배 인상 ($0.50/$3.00 → $1.00/$6.00):
  1교환 비용: $0.004 → $0.008
  CPC $0.05 기준 소진 click 순이익: $0.050 - $0.016 = +$0.034
  → 여전히 흑자

대안: Gemini 2.5 Flash ($0.15/$0.60) 또는 Flash-Lite ($0.10/$0.40)
  → 현재 대비 3~5배 저렴, 품질 약간 하락
```

```
구독자 비용: 30교환/월 × $0.004 = $0.12 + daily_fortune $0.18 = $0.30/월
구독자 이익: $3.50 - $0.30 = +$3.20/월 (수익률 1,067%)

무료 유저 30일 이익: +$0.055 × 30 = +$1.65
구독자 30일 이익: +$3.20

→ 구독 전환 1명 = 무료 유저 2명 분량
→ 전환율 2%만 달성해도 수익 크게 증가
```
