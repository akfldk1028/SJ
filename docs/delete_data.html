데이터 삭제 요청 방법

  1. 아래 이메일 주소로 데이터 삭제 요청 메일을 보내주세요.
  2. 메일 제목에 "[사담] 데이터 삭제 요청"이라고 작성해주세요.
  3. 본문에 앱에서 사용한 이름 또는 생년월일을 기재해주세요.
  4. 요청 접수 후 7영업일 이내에 삭제를 완료하고 확인 메일을 보내드립니다.
